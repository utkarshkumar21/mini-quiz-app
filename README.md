Mini Quiz Web Application

A front-end web application simulating an online quiz with multiple-choice questions and dynamic scoring. Built using HTML, CSS, and JavaScript, demonstrating interactive UI and client-side CRUD concepts.

Features

Interactive multiple-choice quiz with real-time score calculation

Responsive, clean, and user-friendly interface

Client-side management of questions and user responses

Tech Stack

HTML, CSS, JavaScript

Concepts: DOM manipulation, dynamic UI, CRUD

How to Run

Clone the repository:

git clone https://github.com/your-username/mini-quiz-app.git 


Open client/index.html in a browser

Attempt the quiz and view your score
